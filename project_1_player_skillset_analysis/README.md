### Football Player Skillset Analysis

**Objective:**
Analyze and quantify a football player’s skillset using event-level data, and statistically compare their performance to peers in a major international tournament to support data-driven player evaluation.

**Data:**
- Event-level football data from StatsBomb
- Actions performed during competitive international matches

**Methodology:**
- Selection of key on-ball actions relevant to the player’s role
- Feature engineering based on event characteristics
- Statistical comparison against other players in the tournament
- Visualization of actions to support qualitative interpretation

**Results:**
- Identification of key strengths in the player’s action profile
- Comparison with tournament peers to contextualize performance

**Interpretation:**

The analysis demonstrates how event-level data can be used to profile player skillsets and support scouting-style evaluations.
